# Critter Switch

Neon-styled single-page prototype for **Kobold Detektor / Critter Switch**.

## Quickstart

Open `index.html` directly in a browser, or run a static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Project structure

- `index.html` – App layout and semantic structure
- `styles.css` – Visual theme, spacing, component styling
- `app.js` – Lightweight interaction logic (scan/start + tab switching)

## Product design artifacts

Derived planning artifacts for the Critter Switch MVP are available in `docs/`:
